# License Compliance for Bundled Fonts

This document details the open-source fonts bundled with the JPE Sims 4 Mod Translator application and their respective licenses.

## Overview

The application includes a comprehensive collection of open-source fonts that are freely distributable under their respective open-source licenses. All fonts have been selected to ensure compliance with their licensing terms.

## Font Licenses

### Apache 2.0 License
- **Fonts**: Roboto, Roboto Mono, Roboto Slab, Cousine
- **Details**: 
  - Allows commercial use, modification, distribution, and patent use
  - Requires attribution in documentation
  - Does not require same license for derivative works
  - Provides explicit patent protection

### SIL Open Font License (OFL)
- **Fonts**: Noto Sans, Merriweather, Source Serif Pro, Crimson Text, PT Serif, Inconsolata, Montserrat, Playfair Display, Raleway, Oswald
- **Details**:
  - Allows commercial use, modification, distribution
  - Requires preservation of copyright notices
  - Allows naming changes for modified versions
  - Prohibits use of reserved font names without permission
  - Does not require same license for derivative works

### Mozilla Public License (MPL)
- **Fonts**: Fira Sans, Fira Code
- **Details**:
  - Allows commercial use, modification, distribution
  - Requires source code availability for modifications
  - Allows relicensing under GPL, LGPL, or MPL
  - Preserves patent rights

### Ubuntu Font License
- **Fonts**: Ubuntu, Ubuntu Mono
- **Details**:
  - Allows personal and commercial use
  - Permits modification and distribution
  - Requires clear attribution
  - Maintains copyright notices

### Adobe Open Source License
- **Fonts**: Source Sans Pro, Source Code Pro
- **Details**:
  - Allows commercial use, modification, distribution
  - Requires preservation of copyright notices
  - Does not require same license for derivative works
  - Provides patent protection

## Compliance Requirements

To ensure full compliance with the licensing terms:

1. Attribution is provided in this document and the main README
2. Copyright notices are preserved in the font files where applicable
3. The fonts are distributed under their respective open-source licenses
4. No restrictions are placed on the fonts beyond those specified in their licenses

## Installation Rights

The bundled font installer allows users to install fonts to their system under the rights granted by the respective open-source licenses. The application does not modify, redistribute, or restrict the fonts beyond their original license terms.
