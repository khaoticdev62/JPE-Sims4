## ğŸš€ **Setup Instructions:**

### **Prerequisites:**
1. Node.js (v16 or higher)
2. npm or yarn package manager
3. Android Studio (for Android development)
4. (Optional) Xcode if building for iOS on Mac later

### **Initial Setup:**
```bash
# Navigate to the mobile app directory
cd mobile_app

# Install dependencies
npm install

# Install React Native CLI globally
npm install -g react-native-cli

# For Android development, start the emulator
# Or connect a physical Android device in developer mode
```

### **Running the App:**
```bash
# For Android
npx react-native run-android

# For iOS (requires Mac)
npx react-native run-ios
```

- `/api/v1/translate` - Translation endpoints
- `/api/v1/build` - Build execution
